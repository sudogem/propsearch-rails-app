# propsearch-rails-app backend

## Development Environment Setup

- NOTE: All commands should be ran in the root directory unless otherwise specified

**DB Migration**
Setup the db migration and seed
```
$ cd backend
$ rails db:migrate
$ rails db:seed
```

**Run the backend:**
```
$ cd backend
$ rails s
```

**Running the test**  
Run the rspec test
```
$ cd backend
$ bundle exec rspec
```
